# Tahanan

A simple website using HTML, CSS and Javascript

## Run Locally

Clone the project

```bash
  git clone https://github.com/MrXInfinity/Tahanan.git
```

Go to the public directory

```bash
  cd Tahanan
```

Install Live Server extension on Visual Studio Code and click `Go Live`
